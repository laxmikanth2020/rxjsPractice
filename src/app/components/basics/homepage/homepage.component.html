1. What is RXJS?

    2. RXJS is the JavaScript implementation of the Reactive Extension API. RXJS provides us single API to deal the data stream that is coming from the multiple sources over the time by using observable patterns.

  2. What are the different types of Data Streams handle with RXJS?
    • Synchronous Data ----- Arrays, Objects, etc
    • Asynchronous Data ----- SetTimeOut(), SetInterval()
    • Getting data from server as http calls.
    • DOM events
    • Functions with return values
  3. How Observer register with Observable in RCJS?

    4. In RXJS the observable having the subscribe() method, by using this method the observer will register to the observable to get the asynchronous data.

  4. How observable pass the data to observer in RXJS?

    5. while observer register to the observable using subscribe() method, the observer passes the object parameter to the subscribe() method. The object contains three methods next(), error(), and complete() methods.

    • whenever the observable wants to send the data to observer it calls the next() method that is provided by the observer.
    • When the error occurs the observable calls the observer error() method.
    • Finally the Observable call the complete() method when it is done sending the values.
